# Sistema-de-gerenciamento-de-garagem
Sistema de Gerenciamento de Estacionamento desenvolvido para controlar a entrada, permanência e saída de veículos de forma organizada e eficiente. O projeto foi desenvolvido utilizando HTML, CSS, JavaScript, AJAX, PHP e MySQL, oferecendo funcionalidades como cadastro de veículos, registro de entradas e saídas, cálculo automático do tempo e etc.
